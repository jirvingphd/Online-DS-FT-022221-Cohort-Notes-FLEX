# Data Science Curriculum

Welcome! This site hosts cohort study group notes from a Flatiron School Data Science bootcamp, taught by **James Irving, PhD**.

The curriculum covers the full data science stack — from Python fundamentals through deep learning and deployment — across 5 phases.

## Who This Is For

- **Bootcamp students** looking for a reference while working through assignments
- **Self-learners** who want structured, example-driven data science notes
- **Anyone** who wants to see real teaching notes from an intensive DS program

## How to Use This Site

- Browse topics using the sidebar on the left
- Each page is a rendered Jupyter notebook with code, outputs, and explanations
- Click the **rocket icon** (🚀) at the top of any page to open it interactively in Google Colab or Binder

## Curriculum Overview

| Phase | Topics |
|---|---|
| **Phase 1** — Python & Data Fundamentals | Getting started, Bash/Git, Control flow, NumPy/Pandas, Data cleaning, Visualization, SQL, NoSQL, APIs, Web scraping |
| **Phase 2** — Statistics & Regression | Probability, Distributions, CLT, Hypothesis testing, ANOVA, A/B testing, Bayesian stats, Linear & Multiple Regression |
| **Phase 3** — Machine Learning | OOP, Linear algebra, Gradient descent, Regularization, Logistic regression, KNN, Decision trees, Ensemble methods, SVMs |
| **Phase 4** — Advanced ML & Time Series | PCA, Clustering, PySpark, Recommendation systems, Time series, NLP, Neural networks, AWS |
| **Phase 5** — Capstone & Bonus Topics | Capstone kickoff, APIs, Streamlit, Tableau, Plotly |
